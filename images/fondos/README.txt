CARPETA PARA FONDOS
===================

Coloca aquí las imágenes de fondo para usar en la página:

- fondo-hero.jpg (opcional) - Fondo para la sección hero
- fondo-seccion.jpg (opcional) - Fondos para otras secciones

Recomendaciones:
- Formatos: JPG, PNG
- Tamaño recomendado: 1920x1080px o superior
- Optimiza las imágenes antes de subirlas
- Si usas fondos, actualiza las rutas en styles.css
