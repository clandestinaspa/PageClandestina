CARPETA PARA FOTOS DE MODELOS
==============================

Coloca aquí todas las fotos profesionales:

IMÁGENES REQUERIDAS:
- foto-profesional.jpg - Foto principal para la sección "Sobre Mí"
  Tamaño recomendado: 400x500px o proporción similar (vertical)
  Formato: JPG, PNG o WebP

- foto-1.jpg hasta foto-6.jpg - Fotos para la galería
  Tamaño recomendado: 400x500px o proporción similar (vertical)
  Formato: JPG, PNG o WebP
  Puedes agregar más fotos (foto-7.jpg, foto-8.jpg, etc.) y actualizar el HTML

RECOMENDACIONES:
- Optimiza las imágenes antes de subirlas (reduce el tamaño del archivo)
- Usa nombres descriptivos si lo prefieres, pero actualiza las rutas en index.html
- Formatos soportados: JPG, PNG, WebP
- Tamaño máximo recomendado por imagen: 500KB-1MB

Para agregar más fotos a la galería:
1. Agrega la imagen en esta carpeta
2. Edita index.html y agrega un nuevo <div class="gallery-item"> con la ruta de tu imagen
