CARPETA PARA LOGO
=================

Coloca aquí tu logo principal:

- logo.png (o logo.jpg) - Logo principal para la navegación
  Recomendado: formato PNG con fondo transparente
  Tamaño sugerido: altura de 40-60px, ancho proporcional

- favicon.png (opcional) - Icono pequeño para la pestaña del navegador
  Tamaño: 32x32px o 64x64px
  Formato: PNG o ICO

Nombres de archivo:
- logo.png o logo.jpg (para el logo principal)
- favicon.png (para el icono del navegador)
